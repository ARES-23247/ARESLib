## Quick Setup

ARESLib is optimized for VS Code + AdvantageScope.

1. **Clone** this repository.
2. **Sync** Gradle.
3. **Run Simulator**: `./gradlew runSim`
4. **Deploy**: `./gradlew installDebug`

For detailed setup instructions, mission philosophy, and "The Mountaineer Mindset," visit the [ARESWEB Training Hub](https://ares23247.org/docs/getting-started).
